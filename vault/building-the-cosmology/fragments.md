# Fragments

## Purpose

Defines what actually experiences the Continuity Cycle — separate lives connected to one underlying self.

## Core Principle

Fragments do not share ordinary memory. They experience themselves as separate people, with separate histories, relationships, cultures, bodies, and choices. The reader should feel the connection before understanding it.

## Explanation

Connections between fragments appear through residue, not knowledge: repeated visual motifs, familiar behaviors, emotional recognition without memory, similar gestures or relationship patterns, repeated symptoms, and moments that only become meaningful later through comparison.

This is a deliberate constraint, not an oversight. If fragments shared memory or knew their own connection, the story would collapse into exposition — characters explaining a system to each other instead of the reader assembling it from evidence.
