# The Condition

## Purpose

Defines the recurring early-death pattern affecting connected Fragments, and why it must read as biological before it reads as structural.

## Core Principle

The Condition resolves lives before they fully stabilize into continuation. It is not punishment. It is normal system behavior inside the hidden architecture — a pattern characters suffer before anyone understands what it is.

## Explanation

The Condition isn't tied to one universal calendar age; different worlds measure time, maturity, and biology differently. The consistent rule is emotional and structural, not numeric. Its visible sign is the loss of eye color, with the exact appearance depending on each Fragment's original eyes.

**Resolution** is the normal process by which the Condition completes — for these Fragments, that usually means early death. The story should never frame this as evil destiny. It's a pattern, not a verdict, and its deeper continuity function should only become clear through later evidence, never stated upfront.
